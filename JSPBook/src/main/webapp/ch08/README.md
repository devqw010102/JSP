## 유효성 검사

#### 유효성 검사의 종류
| 유효성 검사 | 설명 |
| ---------- | ------|
| 기본 유효성 검사 | 폼 페이지에 입력된 데이터 값의 존재 유무를 검사 |
| 데이터 형식 유효성 검사 | 폼 페이지에 입력된 데이터 값이 특정 패턴에 적합한지 여부를 검사하며 정규 표현식을 사용 |


### 정규 표현식
<pre>
  특정한 규칙을 가진 문자열의 집합을 표현하는 데 사용하는 형식 언어
</pre>

###### 자주 사용되는 패턴
| 데이터 형식 | 패턴 |
| ----------- | ---- |
| 숫자만 | ^[0-9]*$ |
| 영문자만 | ^[a-zA-Z]*$ |
| 한글만 | ^[가- ]*$ |
| 영문자와 숫자만 | ^[a-zA-Z0-9]*$|
